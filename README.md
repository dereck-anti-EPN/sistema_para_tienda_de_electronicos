                                              SISTEMA DE GESTION PARA TIENDA DE ELECTRONICOS

Este es un programa echo con python que sirve como caja registradora para una tienda de electronicos, usted puede: registrar, mostrar y eliminar un pedido. 
Este programa tiene un menu simple: 
En la opcion de registrar podra regitrar un pedido nuevo con los datos del cliente: nombre, apellido, cedula, direccion, tambien puede poner la opcion que el cliente quiso comprar.
Si se pone de la opcion de mostrar los pedidos se mostraran todos los pedidos registrados de todos los clientes junto con sus datos,el pago total con IVA y sin IVA, y un codigo de pedido que se crea automaticamente, en el caso de no existir ningun registro se mostrara que no existe niguno.
Si se pone la opcion de mostrar un pedido, se pedira un codigo de pedido para que se muestre el pedido registrado en especifico y mostrara todos los datos, de no existir el codigo de pedido muestra un mensaje de error.
Si se elije la opcion de eliminar un registro se pedira el codigo de pedido y se eliminara, si no existe un registro con el codigo ingresado mostrara un mensaje de error.
